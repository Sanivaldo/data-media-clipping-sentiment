Engenharia de Dados com Azure

Projeto com foco em ingestão, transformação e visualização de dados públicos

Sobre o Projeto

Este projeto demonstra uma arquitetura de dados moderna utilizando serviços da Microsoft Azure, incluindo Data Factory, Storage, Databricks e Power BI. O objetivo é processar dados públicos brasileiros com qualidade e gerar insights com valor social.

Sobre Mim

Sou Engenheiro de Dados com mais de 5 anos de experiência na área de dados, atuando em projetos de análise, transformação e arquitetura de soluções em nuvem. Ao longo da minha trajetória, desenvolvi pipelines de dados robustos, modelos analíticos, integrações entre sistemas e dashboards voltados para apoio à decisão estratégica.

Possuo certificações oficiais da Microsoft, incluindo:
PL-300: Power BI Data Analyst Associate
DP-900: Azure Data Fundamentals
(atualmente em preparação para a AZ-900 e DP-600)

Desenvolvi, do zero, arquiteturas completas na plataforma Microsoft Azure, integrando serviços como Data Factory, Blob Storage, Azure SQL Database, Databricks (PySpark) e Power BI para gerar valor com dados reais da sociedade brasileira.

Minha missão é aplicar tecnologia com propósito — traduzindo dados complexos em soluções que realmente importam.
