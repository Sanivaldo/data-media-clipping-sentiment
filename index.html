Engenharia de Dados com Azure

Projeto com foco em ingestão, transformação e visualização de dados públicos

Sobre o Projeto

Este projeto demonstra uma arquitetura de dados moderna utilizando serviços da Microsoft Azure, incluindo Data Factory, Storage, Databricks e Power BI. O objetivo é processar dados públicos brasileiros com qualidade e gerar insights com valor social.
